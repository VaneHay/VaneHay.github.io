Java 基础语法

Java 程序是一系列对象的集合，对象与对象之间互相调用对象的方法来协同工作。

  • 对象 用 new 类出来的实例，有 属性和方法
  • 类 描述一些对象的属性和方法
  • 方法 逻辑运算，数据修改 在方法中完成
  • 实例变量

注：.java后缀，类名与文件名一致，否则会报错

修饰符

  • 访问控制修饰符 default public protected private
  • 非访问控制修饰符 final abstract static synchronized

变量

  • 局部变量
  • 类变量 静态变量
  • 成员变量 非静态变量
