抽象类

  • 定义为抽象方法，这个类也是抽象类。
  • 抽象方法只定义调用方式，不做任何功能逻辑。比如：abstract public function get_userinfo($id);
  • 定义为抽象的类不能被实例化 (不能 new 是实例化)
  • 继承一个抽象类的时候，子类必须定义父类中的所有抽象方法，所需参数数量必须一致
  • 继承抽象类时访问控制必须和父类中一样（或者更为宽松）
