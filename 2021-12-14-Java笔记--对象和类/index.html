Java 对象和类

变量

  • 局部变量 方法中申明或初始化，用完销毁。
  • 成员变量 类中，但是是方法体外申明的变量。比如：string name
  • 类变量 静态变量，static 类型

构造方法

构造方法的方法名必须与类名同名，一个类可以有多个构造方法

创建对象

包（package） 与 引入（import）

如果源文件包含 import 语句，那么应该放在 package 语句和类定义之间。如果没有 package 语句，那么 import 语句应该在源文件中最前面。
